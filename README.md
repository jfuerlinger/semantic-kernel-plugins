# Semantic Kernel Plugins Showcase

This repository is a showcase of Semantic Kernel plugins.

## Getting Started

To start the application, ensure you have a `secrets.json` file with the following content:

```json
    "OpenAi": {
        "DeploymentName": "---",
        "Endpoint": ""---",",
        "ApiKey": ""---",",
        "serviceID": ""---","
    }
```
